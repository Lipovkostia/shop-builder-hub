
## Фильтры для таблицы прайс-листа

### Что будет добавлено
Рядом с полем "Поиск товаров..." появится строка фильтров, позволяющих быстро находить нужные товары в прайс-листе:

1. **Фильтр по категории** -- выпадающий список с категориями магазина. Позволяет показать товары только из выбранной категории (главной или подкатегории).

2. **Фильтр по статусу** -- выпадающий список со значениями:
   - Все статусы (по умолчанию)
   - В наличии
   - Предзаказ
   - Нет в наличии
   - Скрыт

3. **Фильтр "С ценой / Без цены"** -- позволяет найти товары, у которых ещё не установлена цена (полезно после импорта).

4. **Кнопка сброса фильтров** -- появляется, когда хотя бы один фильтр активен.

### Расположение
Фильтры будут размещены в одной строке с поиском (с переносом на мобильных), компактными выпадающими списками (Select).

### Технические детали

**Файл: `src/pages/AdminPanel.tsx`**

- Добавить три новых состояния:
  - `catalogFilterCategory: string` (ID категории или пустая строка)
  - `catalogFilterStatus: string` (статус или пустая строка)
  - `catalogFilterPrice: string` ("all" | "with_price" | "no_price")

- Сбрасывать фильтры при выходе из прайс-листа (в обработчике кнопки "Назад")

- Расширить блок фильтрации товаров (строки 4812-4814) -- добавить проверку по категории, статусу и наличию цены к существующему `catalogProductSearch`

- В область поиска (строки 4621-4629) добавить компоненты Select для каждого фильтра + кнопку сброса

Логика фильтрации:
```text
products
  .filter(in catalog)
  .filter(matches search text)
  .filter(matches category filter)
  .filter(matches status filter)
  .filter(matches price filter)
```
