

## Экспорт прайс-листа в PDF

### Что будет сделано

Добавлю кнопку "Скачать PDF" в панели прайс-листа и диалог настроек перед выгрузкой, аналогичный существующему диалогу Excel-экспорта. Пользователь сможет:
- Выбрать, какие столбцы включать в PDF
- Включить/выключить мини-фотографии товаров
- Скачать готовый PDF-файл

PDF будет генерироваться прямо в браузере с помощью библиотеки `jspdf` + `jspdf-autotable` (для таблиц).

### Формат PDF

Каждый товар -- строка таблицы. Если включены фотографии, первая колонка содержит миниатюру (примерно 30x30px). Далее идут выбранные столбцы (название, цена и т.д.). Заголовок документа -- название прайс-листа.

### Новые файлы

**1. `src/components/admin/CatalogPdfExportDialog.tsx`**
- Диалог настроек PDF-экспорта

**2. `src/lib/pdfExport.ts`**
- Функция `exportCatalogToPdf(catalogName, products, enabledColumns, includePhotos)`
- Использует `jspdf` для создания PDF
- Использует `jspdf-autotable` для рендера таблицы
- Если включены фото -- загружает изображения, конвертирует в base64, вставляет как миниатюры в ячейки таблицы
- Формат: A4, landscape (для удобства), шрифт с поддержкой кириллицы

### Технические детали

- **Новые зависимости:** `jspdf` и `jspdf-autotable`
- **Кириллица в PDF:** jspdf по умолчанию не поддерживает кириллицу. Будет использован встроенный механизм через подключение шрифта (Roboto или подобный) в base64, либо альтернативный подход -- генерация PDF через HTML-рендеринг (`jspdf.html()`)
- **Загрузка фото:** Изображения загружаются через fetch, конвертируются в base64 через canvas, и вставляются в таблицу. Для товаров без фото ячейка остаётся пустой
- **Размер миниатюр:** ~25x25px в PDF-таблице
- **Интерфейс CatalogExportProduct** уже содержит поле `images: string[] | null` -- будет использовано для получения URL первого фото
