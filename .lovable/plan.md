## Что делаем

Добавляем удобную панель управления для вкладки «Товары для Авито» в разделе Авито. Главные цели:

1. Группировать (категоризировать) объявления внутри таблицы по своим папкам с произвольными названиями.
2. Чётко визуально разделить колонки, которые уходят в выгрузку XLSX/ZIP для Авито, и колонки, которые используются только для внутренней фильтрации/порядка.
3. Сохранить текущую логику выгрузки без изменений — меняется только UI и хранение групп.

## Как это будет выглядеть

### 1. Боковая панель «Мои категории» (слева от таблицы)

- Список своих папок-групп, например «Сыры», «Масла», «Рыба», «Без категории».
- Кнопки: + Новая группа, переименовать, удалить, перетаскивание для сортировки.
- Счётчик товаров в каждой группе и индикатор ошибок (красная точка).
- Клик по группе фильтрует таблицу → показываются только товары этой группы.
- Пункт «Все товары» сверху + «Без группы» снизу.

### 2. Назначение группы товару

- В каждой строке таблицы — новый компактный селектор «Группа» (бейдж со стрелкой). Клик открывает список ваших групп + «Создать новую».
- Массовое назначение: выделить чекбоксами несколько строк → в верхней панели появляется кнопка «Переместить в группу…».
- Drag-and-drop строки на группу в сайдбаре (опционально, как ускоритель).

### 3. Цветовая разметка колонок

Вводим два класса колонок и подсвечиваем их и в шапке, и фоном ячеек:

- Экспортные (идут в XLSX для Авито): мягкий зелёный акцент в шапке + маленькая иконка «облако/выгрузка» в заголовке. Тултип: «Поле уходит в выгрузку Авито».
- Служебные (только для фильтрации/порядка, в выгрузку не идут): нейтрально-серый/синеватый фон шапки + иконка «фильтр». Тултип: «Внутреннее поле, в выгрузку не попадает».

Колонка «Группа» — служебная (не в выгрузке).
Все существующие колонки, которые сейчас попадают в Excel/ZIP, помечаются как экспортные. Логика самой выгрузки не меняется.

Над таблицей — компактная легенда: два цветных чипа «В выгрузке» / «Только для фильтрации», плюс переключатель «Скрыть служебные колонки» для чистого вида при проверке выгрузки.

### 4. Панель управления над таблицей

Тонкая sticky-панель с:
- Чипами активных фильтров (категория Авито, ошибки, группа) с крестиками.
- Кнопками: «Развернуть/свернуть все группы», «Скрыть служебные колонки», «Сбросить фильтры».
- Счётчиками: всего / отфильтровано / выбрано / с ошибками.

### 5. Режим «по группам» в самой таблице (опционально, по умолчанию вкл.)

- Строки группируются заголовками-разделителями с названием группы, счётчиком и кнопкой свернуть/развернуть.
- Виртуализация сохраняется (через @tanstack/react-virtual, как сейчас).

## Хранение

Новая таблица `avito_product_groups` (id, store_id, name, sort_order, color) и `avito_product_group_items` (group_id, product_id) с RLS по магазину. Это никак не влияет на структуру выгрузки.

## Что НЕ меняется

- Поля и формат XLSX/ZIP выгрузки Авито.
- Логика ошибок, AI-описаний, AI-категорий, импорта Excel.
- Вкладки «Объявления с ошибками» и «Активные объявления» (по желанию потом распространим тот же сайдбар туда же).

## Технические детали

- Файлы UI: расширяем `src/components/admin/AvitoSection.tsx` (вкладка `feed`), добавляем компоненты `AvitoGroupsSidebar.tsx`, `AvitoGroupBadge.tsx`, `AvitoColumnLegend.tsx`.
- Состояние групп: хук `useAvitoProductGroups(storeId)` с CRUD + назначениями, оптимистичные обновления, как в `useProductGroups`.
- Колонки помечаем флагом `kind: "export" | "internal"` в конфиге; цветовые токены добавим в `index.css` (`--avito-export`, `--avito-internal`) — только HSL, через семантические токены.
- Миграция БД: создание таблиц + GRANT + RLS (доступ только владельцу магазина / админам магазина).
- Никаких изменений в edge-функциях экспорта.

## Порядок реализации

1. Миграция БД + хук `useAvitoProductGroups`.
2. Сайдбар групп и фильтрация таблицы по выбранной группе.
3. Колонка/бейдж «Группа» в строке + массовое перемещение.
4. Цветовая разметка колонок + легенда + тумблер «Скрыть служебные».
5. Группировка строк заголовками (с виртуализацией) и сворачивание.
6. Проверка, что выгрузка XLSX/ZIP осталась идентичной.
