

## Добавить опцию "Не скрывать товары, которых нет в прайс-листе"

### Проблема
При импорте Excel через AI-помощника система автоматически скрывает все товары, которых нет в загружаемом файле. Это неудобно, если вы хотите просто обновить цены на часть товаров, не трогая остальные.

### Решение
Добавить переключатель (чекбокс) в интерфейс импорта, который позволяет выбрать:
- **Скрыть отсутствующие** (текущее поведение, по умолчанию) -- товары, которых нет в Excel, будут скрыты из прайс-листа
- **Не скрывать** -- обновляются только те товары, которые есть в файле, остальные не затрагиваются

Чекбокс будет отображаться на этапе маппинга колонок (перед началом импорта).

### Технические изменения

**1. `src/lib/priceListImport.ts`**
- Добавить параметр `hideNotInFile: boolean` в функцию `importProductsToCatalogExtended`
- Обернуть блок скрытия товаров (строки 846-869) в условие `if (hideNotInFile)`

**2. `src/components/admin/AIAssistantPanel.tsx`**
- Добавить состояние `hideNotInFile` (по умолчанию `true` для обратной совместимости)
- Добавить чекбокс "Скрыть товары, которых нет в файле" рядом с маппингом колонок
- Передать параметр в `startImport` и далее в `importProductsToCatalogExtended`

### Пользовательский опыт
1. Загружаете Excel файл в AI-помощник
2. На экране маппинга колонок появляется чекбокс "Скрыть товары, которых нет в файле" (включен по умолчанию)
3. Снимаете галочку, если хотите только обновить цены без скрытия остальных товаров
4. Продолжаете импорт как обычно
