

# Вкладка "Витрина": поле телефона + блок контактов на витрине

## Что будет сделано

1. В админке на вкладке "Витрина" — поле ввода номера телефона (сохраняется в `showcase_phone`)
2. На публичной витрине под шапкой — тонкий блок с кликабельным номером телефона
3. На мобильных при скролле вниз блок плавно скрывается, при скролле вверх — плавно появляется

## Технические детали

### 1. Админка — вкладка "Витрина" (`AdminPanel.tsx`)

Заменяем заглушку на форму с полем "Номер телефона":
- Input для ввода номера
- Кнопка "Сохранить"
- Сохранение в поле `showcase_phone` таблицы `stores` (поле уже существует в БД)

### 2. Публичная витрина — блок контактов

Пока публичная витрина ещё не создана, но блок будет подготовлен как отдельный компонент `ShowcaseContactBar.tsx`:
- Тонкая полоска под шапкой (`py-1.5`, `text-xs`)
- Кликабельный номер телефона (`tel:` ссылка)
- Иконка телефона слева
- Отображается только если `showcase_phone` заполнен

### 3. Поведение при скролле (мобильная версия)

- Отслеживание направления скролла через `useEffect` + `scroll` event
- При скролле вниз: `transform: translateY(-100%)` с `transition-transform duration-300`
- При скролле вверх: `transform: translateY(0)` — блок плавно возвращается
- На десктопе блок всегда виден
