

## Реструктуризация главной страницы: 3 колонки с интерактивным каталогом

### Новый макет

Страница будет разделена на 3 вертикальные колонки (на десктопе):

```text
+---------------------------+---------------------------+---------------------------+
|   4 инфо-блока (UTPs)     |                           |  Продавец / Покупатель    |
+---------------------------+   Интерактивное табло      +---------------------------+
|                           |   "Мой каталог"           |                           |
|   Список товаров          |   (сюда переносятся       |   Форма входа /           |
|   (с чекбоксами)          |    выбранные товары)      |   регистрации             |
|                           |                           |                           |
|   [x] Сыр Бри...          |   Сыр Бри...              |                           |
|   [ ] Сыр Камамбер...     |   Пармезан...             |                           |
|   [x] Пармезан...         |                           |                           |
|                           |                           |                           |
|   [ Добавить в каталог ]  |   [ Очистить ]            |                           |
+---------------------------+---------------------------+---------------------------+
```

### Что будет сделано

1. **Левая колонка** -- список товаров (LandingProductTable) с чекбоксами для выбора, кнопка "Добавить в каталог" снизу или сверху. Над списком -- 4 инфо-блока (UTPs) в сетке 2x2.

2. **Средняя колонка** -- интерактивное табло "Мой каталог" (новый компонент `LandingDemoCart`). Выбранные товары из левого списка плавно появляются здесь с анимацией. Можно удалять товары из этого блока. Это демо-каталог для демонстрации функционала платформы (данные хранятся только в localStorage, без бэкенда).

3. **Правая колонка** -- существующий блок авторизации (Продавец/Покупатель), без изменений.

### Технические детали

**Файлы, которые будут изменены:**

- `src/pages/Index.tsx` -- переделка макета с `grid-cols-3` на десктопе, добавление стейта для выбранных товаров и передача в компоненты.

- `src/components/landing/LandingProductTable.tsx` -- добавление чекбоксов к каждой строке, кнопка "Добавить в каталог", колбэк для передачи выбранных товаров наверх. Инфо-блоки (UTPs) останутся над таблицей в этой же колонке.

- `src/components/landing/LandingDemoCart.tsx` (новый) -- компонент интерактивного табло. Отображает добавленные товары в компактном табличном виде (как прайс-лист). Поддерживает анимацию появления (`animate-fade-in`), удаление отдельных товаров, кнопку "Очистить". Состояние сохраняется в localStorage.

**Анимация переноса:**
- При нажатии "Добавить в каталог" выбранные товары появляются в среднем блоке с плавной анимацией `animate-fade-in` (уже есть в проекте).

**Адаптивность:**
- На мобильных устройствах колонки располагаются вертикально: инфо-блоки, список товаров, демо-каталог, форма авторизации.
