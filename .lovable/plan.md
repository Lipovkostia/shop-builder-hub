
# План: Компактный и удобный интерфейс сопоставления колонок

## Проблема
Окно сопоставления колонок имеет слишком много фиксированных областей:
- Шапка с именем файла (header) — фиксирована сверху
- Предпросмотр данных 120px — фиксирован снизу  
- Кнопки действий — фиксированы в самом низу

В результате центральная прокручиваемая область очень мала и неудобна для работы.

## Решение

### 1. Уменьшить отступы везде в 2 раза
- `px-6` → `px-4`
- `py-4` → `py-2` 
- `space-y-6` → `space-y-3`
- Внутренние `space-y-3` → `space-y-2`
- `pl-8` → `pl-6`

### 2. Сделать header компактнее
- Уменьшить padding: `py-4` → `py-2`
- Уменьшить размер иконки файла
- Убрать градиентный фон или сделать его менее высоким

### 3. Уменьшить или скрыть preview снизу
- Уменьшить высоту preview с 120px до 80px
- Или сделать preview сворачиваемым
- Или показывать preview inline вместе с select'ами

### 4. Компактнее RadioGroup и Select'ы
- Уменьшить padding у radio-кнопок: `p-2.5` → `p-1.5`
- Сделать Select'ы менее высокими
- Уменьшить gap между элементами

### 5. Компактнее чекбоксы для полей обновления
- Уменьшить отступы между label и description
- Использовать inline layout где возможно

## Технические изменения

### ExcelColumnMapping.tsx:

**Header (строки 222-235):**
```
- px-6 py-4 → px-4 py-2
- w-10 h-10 → w-8 h-8  
- h-5 w-5 → h-4 w-4
```

**Основной контент (строки 237-416):**
```
- px-6 py-4 space-y-6 → px-4 py-2 space-y-3
- Внутренние space-y-3 → space-y-2
- pl-8 → pl-6
- p-2.5 → p-1.5 (для radio buttons)
- w-6 h-6 → w-5 h-5 (для иконок шагов)
```

**Preview section (строки 419-469):**
```
- h-[120px] → h-[80px] или сделать сворачиваемым
- px-6 py-3 → px-4 py-2
```

**Footer с кнопками (строки ~470-484):**
```
- px-6 py-3 → px-4 py-2  
- gap-3 → gap-2
```
